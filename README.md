# 🧠 GitHub Repository Explainer
The GitHub Repository Explainer is a comprehensive tool designed to analyze and provide insights into GitHub repositories. It offers a range of features, including repository inspection, file selection, downloading, and explanation generation. The project is built using a combination of technologies, including Express.js, React, and various utility libraries.

## 🚀 Features
* Analyze GitHub repositories and provide insights into their structure and content
* Inspect repositories, select important files, and download them for further analysis
* Generate explanations for the analyzed repositories
* Utilize server-sent events to provide real-time updates to clients
* Implement a custom React hook for managing the analysis process
* Include a service function for establishing connections to the server-side API

## 🛠️ Tech Stack
* Frontend: React.js, Tailwind CSS
* Backend: Express.js, Node.js, Groq SDK, Github REST API
* Database: 
* AI Tools: Groq API
* Build Tools: 
* Utilities: dotenv, p-limit, Helmet, Morgan, CORS

## 📦 Installation
To install the project, follow these steps:
1. Clone the repository using `git clone`
2. Install the required dependencies using `npm install`
3. Set up the environment variables using `dotenv`
4. Start the server using `node server.js`
5. Start the client using `npm start`

## 💻 Usage
To use the project, follow these steps:
1. Open the client-side application in a web browser
2. Enter the GitHub repository URL in the input field
3. Click the "Explain" button to start the analysis process
4. The application will display the Explanation in real-time
